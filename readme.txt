Generate a Lorem Ipsum placeholder by the number of words

== Description ==
Generate a Lorem Ipsum placeholder by the number of words. You can use the [lorem words=XXX] shortcut to generate the specific amount of words. For example: If you want to generate a placeholder with 300 words you can use [lorem words=300]. Have fun.
== Installation ==
Upload the all files into your ‘wp-content/plugins’ directory
Activate the plugin inside your WordPress Backend
Use [lorem words=XXX] shortcode in any kind of post to generate the placeholder text.
